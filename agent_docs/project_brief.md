# Project Brief (Persistent)

- **Product Vision:** A zero-planning, one-stop-shop travel app that does the heavy lifting up front with a clean UI, providing a verifiable itinerary and booking links.
- **Design Feel:** Modern, immersive, glassmorphic, and nature-inspired. The UI floats above the content.
- **Target Audience:** Travelers of all ages wanting spontaneous getaways without spreadsheet planning.
- **Budget Constraint:** Strictly utilize free tiers. Manage Google Places API aggressively to avoid bill shock.
- **Workflow Expectation:** The AI must explain the architectural "why" before writing the code so the developer learns the system.
